# eating-medication

轻量级的用药提醒工具，支持定时提醒和用药记录。

## 功能

- 添加药物和提醒时间
- 定时提醒服药
- 记录服药历史
- 自动检查 GitHub 最新版本并提示更新

## 使用

```bash
python main.py add       # 添加药物
python main.py list      # 列出药物
python main.py take      # 记录服药
python main.py records   # 查看记录
python main.py remind    # 启动提醒
```
